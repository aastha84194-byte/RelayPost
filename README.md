## ✨ Overview

RelayPost is a modern AI-powered news and discussion platform designed to help users discover, explore and engage with trending topics from around the world.

The platform combines clean UI design with intelligent content organization to create a seamless reading experience.

---

## 🌟 Features

✅ Modern Dark UI  
✅ Responsive Design  
✅ Category Based News Filtering  
✅ AI Powered Expert Analysis  
✅ Popular Conversations Section  
✅ Search Functionality  
✅ Authentication System

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/relaypost.git
```

Move into project folder:

```bash
cd relaypost
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

---

## 🔥 Environment Variables

Create a `.env` file in the root directory:

```env
DATABASE_URL=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
OPENAI_API_KEY=
```
